# x86_asm

> *x86 汇编语言： 从实模式到保护模式* linux 版本代码 
> 用 qemu 启动

## usage

* 需要安装 qemu，使用 
  ``` $ qemu-system-i386 -hda <file.bin> ``` 

* or just ``` $ make qemu ```
